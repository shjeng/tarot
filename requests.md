## 요구 사항
- 타로 관련 사이트 
- 셔플 버튼을 누르면 따로 준비해둔 애니메이션(동영상)이 재생되고 영상이 끝나면 카드를 이용자가 선택할 수 있도록 함. 
- 기능은 오늘의 운세와 직접 3장을 뽑아서 AI한테 타로 점을 봐달라고 하는 기능을 만들 예정 
- 카드 이미지는 일단 외부에서 가져오고, 이후에 운영자가 직접 수정할 예정임


## UI, UX 
- 신비로운 분위기의 테마: 다크 모드를 기반으로 한 연한 보라색, 금색 포인트 컬러 사용.
- 모바일 퍼스트: 타로 서비스 특성상 모바일 접속이 압도적이므로 반응형 디자인 필수.

## 기술 스택

- Frontend	React or Next.js + Tailwind CSS	빠른 UI 개발 및 SEO 최적화 유리
- Backend	Node.js (Express) Gemini API 연동 및 비동기 처리 최적화
- AI Integration	Google Gemini Pro API	텍스트 분석 및 운세 생성 엔진
